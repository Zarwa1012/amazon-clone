# Amazon Website Clone (HTML & CSS)

This project is a simple Amazon website clone created using only HTML and CSS.  
It is a front-end practice project designed to improve layout, styling, and basic website structure skills.

## Pages Included

This project contains the following pages:

1. **Home Page (index.html)**  
   - Amazon-style layout  
   - Navigation bar  
   - Basic structure of the website  

2. **Contact Page (contact.html)**  
   - Contact form  
   - User can enter name, email, subject, and message  
   - Styled using Amazon theme colors  

3. **Shop by Categories Page (categories.html)**  
   - Displays different categories of goods  
   - Uses cards/boxes for categories like Electronics, Fashion, Home & Kitchen, etc.  

Each page has its own separate CSS file for styling.
